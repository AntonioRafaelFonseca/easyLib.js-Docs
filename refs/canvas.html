easyLib

Canvas

The easyLib Canvas provides simple functions for drawing on an HTML5 canvas element

Creating a Canvas

To create a canvas, use the createCanvas(width, height, parent) function:

let canvas = createCanvas(500, 500, document.body);

Canvas Functions

createCanvas(width, height, parent)

Creates a canvas element with the specified width and height, and appends it to the parent element.

Parameters:

  • width - The width of the canvas in pixels
  • height - The height of the canvas in pixels
  • parent - The parent DOM element where the canvas will be appended

Example:

let canvas = createCanvas(500, 500, document.body);

fillBackground(color: [r, g, b, a])

Fills the background of the canvas with the specified color using RGB values.

Parameters:

  • color - An array with four values [r, g, b, a] where r, g, b are 0-255 and a is 0-1 opacity

Example:

fillBackground([255, 0, 0, 0.5]);

drawRect(x, y, w, h, borderColor, fillColor)

Draws a rectangle on the canvas with the specified position, size, border color, and fill color.

Parameters:

  • x - The x-coordinate of the top-left corner
  • y - The y-coordinate of the top-left corner
  • w - The width of the rectangle
  • h - The height of the rectangle
  • borderColor - The color of the border (CSS color string)
  • fillColor - The fill color of the rectangle (CSS color string)

Example:

drawRect(50, 50, 100, 100, 'black', 'red');

drawCircle(x, y, radius, borderColor, fillColor)

Draws a circle on the canvas with the specified position, radius, border color, and fill color.

Parameters:

  • x - The x-coordinate of the center
  • y - The y-coordinate of the center
  • radius - The radius of the circle
  • borderColor - The color of the border (CSS color string)
  • fillColor - The fill color of the circle (CSS color string)

Example:

drawCircle(200, 200, 50, 'black', 'blue');

drawLine(x1, y1, x2, y2, borderColor)

Draws a line on the canvas between the specified points with the specified color.

Parameters:

  • x1 - The x-coordinate of the starting point
  • y1 - The y-coordinate of the starting point
  • x2 - The x-coordinate of the ending point
  • y2 - The y-coordinate of the ending point
  • borderColor - The color of the line (CSS color string)

Example:

drawLine(100, 100, 300, 300, 'green');

startShape()

Starts a new custom shape on the canvas. Use this with addCustomPoint() and endShape() to create custom polygons.

Example:

startShape();

addCustomPoint(x, y)

Adds a point to the current shape. Must be called after startShape() and before endShape().

Parameters:

  • x - The x-coordinate of the point
  • y - The y-coordinate of the point

Example:

addCustomPoint(150, 150);

endShape(fillColor, borderColor)

Closes and draws the current shape on the canvas with the specified fill color and border color.

Parameters:

  • fillColor - The fill color of the shape (CSS color string)
  • borderColor - The border color of the shape (CSS color string)

Example:

endShape('orange', 'yellow');

Usage Examples

Example 1: Draw a rectangle

Example:

createCanvas(500, 500, document.body);
fillBackground([255, 255, 255, 1]);
drawRect(50, 50, 100, 100, 'black', 'red');

Example 2: Draw a circle

Example:

createCanvas(500, 500, document.body);
fillBackground([255, 255, 255, 1]);
drawCircle(200, 200, 50, 'black', 'blue');

Example 3: Draw a line

Example:

createCanvas(500, 500, document.body);
fillBackground([255, 255, 255, 1]);
drawLine(100, 100, 300, 300, 'green');

Example 4: Draw a custom shape

Example:

createCanvas(500, 500, document.body);
fillBackground([255, 255, 255, 1]);
startShape();
addCustomPoint(100, 100);
addCustomPoint(150, 50);
addCustomPoint(200, 100);
addCustomPoint(150, 150);
endShape('purple', 'black');
