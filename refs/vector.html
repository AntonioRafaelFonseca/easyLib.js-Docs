easyLib

Vector

The easyLib Vector is a class that represents a vector in 2D space

to define a variable of this class you can use the vector(x, y) function or the new Vector(x, y) constructor

Example:

let yourVector = vector(1, 2);

To acces the x and y values of the vector you can use the yourVector.x and yourVector.y properties

Example:

console.log(yourVector.x);

To add, subtract, multiply and divide vectors you can use the add, sub, mult and div functions

Example:

let vector1 = vector(1, 2);
vector1.add(1, 3);
console.log(vector1);

You can also add, subtract etc. vectors from other vectors with the addVec, subVec, multVec and divVec functions

Example:

let vector1 = vector(1, 2);
let vector2 = vector(5, 2);
vector1.addVec(vector2);
console.log(vector1);

To get the magnitude of the vector you can use the magnitude() function

To get the angle of a vector relative to another vector you can use the angle() function

To get the distance of a vector relative to another vector you can use the distance() function

To get the dot product of a vector relative to another vector you can use the dot() function

To nornalize a vector you can use the normalize() function

Example:

let vector1 = vector(1, 2);
let vector2 = vector(5, 2);
vector1.normalize();
console.log(vector1);
console.log(vector1.magnitude());
console.log(vector1.angle(vector2));
console.log(vector1.distance(vector2));
console.log(vector1.dot(vector2));
